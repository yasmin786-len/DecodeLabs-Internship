# DecodeLabs AI Internship 

## Project 1: Rule-Based AI Chatbot

A chatbot developed using Python dictionaries and control flow. The chatbot responds to predefined user inputs and exits when the user enters "bye" or "exit".

### Technologies Used

* Python
* Dictionaries
* Loops
* Conditional Statements



## Project 2: Data Classification Using AI

A machine learning project using the Iris Dataset and K-Nearest Neighbors (KNN) algorithm.

### Features

* Train-Test Split
* Feature Scaling
* KNN Classification
* Accuracy Evaluation
* Confusion Matrix
* F1 Score

### Technologies Used

* Python
* Scikit-Learn
* NumPy
* Pandas



## Project 3: Recommendation System

A content-based recommendation system that suggests courses based on user interests using TF-IDF Vectorization and Cosine Similarity.

### Features

* User Interest Input
* TF-IDF Vectorization
* Cosine Similarity
* Ranked Recommendations

### Technologies Used

* Python
* Scikit-Learn
